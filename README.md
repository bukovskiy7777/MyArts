# MyArts

View the work of great artists on your smartphone.
Mark your favorites.
Create collections.

Link on [Google Play]

Technologies:
- Single Activity App
- MVVM
- ViewModel
- LiveData
- Paging
- Navigation
- Retrofit2
- Google Auth
